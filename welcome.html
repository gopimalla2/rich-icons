Welcome

Welcome to the G-Collection v2.0, next generation set of icons from IconExperience with a colorful, friendly and professional look. The G-Collection contains more than 1,700 icons which come in eight sizes (16x16, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512), three styles (standard, blue and gradient) and in file formats PNG, ICO and SVG.

Using the Icon Search tool

You can easily access the icons as PNG files by using the Icon Search tool. With this tool, you can search icons by keyword or name and save them to your hard drive.

To start the Icon Search, click "Icon Search" in the menu or click here.

Directly accessing the files

You can also directly access the image files on your computer. In the folders

  • /g_collection/g_collection_png
  • /g_collection/g_collection_svg
  • /g_collection/g_collection_ico
  • /g_collection/g_collection_ico_legacy
you can find the files for each available file type. Each of the folders contains one folder for each of the styles "standard", "blue", and "gradient".

Using the PDF catalog

In the folder /g_collection/g_collection_pdf you will find the PDF catalog of the G-Collection. The catalog contains previews of the icons by category. You can view the catalog with a PDF viewer or print them for offline use.

To open the PDF catalog please click here.

Using the Icon Toolbox

The IconExperience Toolbox will help you to get much more out of the G-Collection. Using the combination feature, you can add any of about 60 badge icons ("add", "delete", "edit", etc.) to any of the G-Collection icons. This way you can create more than 100.000 possible combined icons from the 1700 icons that are included in the G-Collection.

Other features are

  • converting PNG images to GIF or BMP files
  • adding an opaque background and/or changing the foreground color for PNG images
  • adding an empty padding around an icon
  • converting PNG images to hot or disabled state images (used for "classic" icons in Windows XP, Windows Vista or Windows 7 style)

You can find the Toolbox in the folder /toolbox. Please note that in order to run the Toolbox, you must have Java 6 (or higher) installed on your computer.
